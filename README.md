## Usage

1. Install [Anaconda](https://www.continuum.io/downloads)
2. Clone this repository
3. Run the jupyter-notebook (default is enabled by conda) in terminal:

  ```bash
  $ cd ITT-ADA-2017/
  $ jupyter-notebook
  ```

4. Select a workbook and have a try!
